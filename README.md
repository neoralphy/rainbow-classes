# Rainbow Classes

Give every class a unique visual identity in your tabs.

Rainbow Classes assigns deterministic colors to classes, interfaces, traits and enums, making it easier to navigate large codebases with many open files.

Instead of relying only on file names, each class gets a small color badge derived from its fully qualified name. This creates a consistent visual fingerprint across the IDE.

## Features

• deterministic class color hashing  
• tab icon color badges  
• project tree badges  
• multiple palettes  
• works with PHP classes, interfaces, traits and enums  

## Palettes

Rainbow Classes includes several color palettes:

Default  
Balanced colors designed for everyday use.

Pastel  
Soft but still visible colors for a calmer UI.

High Contrast  
Strong colors for maximum readability.

Synthwave  
Retro neon palette inspired by synthwave aesthetics.

## How it works

Each class receives a color derived from a deterministic hash of its fully qualified name.

This means:

• the same class always gets the same color  
• colors remain stable across sessions  
• no configuration is required  

## Supported languages

Currently optimized for:

PHP (PhpStorm / IntelliJ with PHP plugin)

## Settings

You can configure Rainbow Classes under:

Settings → Tools → Rainbow Classes

Options include:

• enable / disable markers  
• palette selection
